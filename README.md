# 设备信息管理系统 python259

本项目是一款集设备、库存信息管理为一体的系统，主要基于Python语言，结合mysql数据库、pyqt5图形用户界面库以及matplotlib数据可视化库进行开发。

## 技术栈

- Python
- MySQL
- PyQt5
- matplotlib

## 功能模块

### 系统管理

- 用户登录

### 数据管理

- 用户信息增删改查
- 备件信息增删改查
- 供应商信息增删改查
- 部门信息增删改查

### 采购管理

- 采购计划增删查
- 采购计划审批功能

### 库存管理

- 备件库存出入库管理
- 备件出入库记录查询
- 备件出入库记录统计柱形图

## 系统角色

- 系统管理员
- 采购人员
- 库存管理人员

## 运行环境

- 操作系统：兼容Windows、Linux等
- Python版本：3.6及以上
- MySQL版本：5.7及以上

## 部署步骤

1. 环境准备：确保Python、MySQL等依赖库正确安装。
2. 数据库配置：根据sql文件建立数据库和表结构。
3. 程序配置：配置程序运行所需的参数和数据库连接信息。
4. 运行程序：执行程序启动脚本，开始运行系统。

device_info_management_system/
